## Project Workflow
1. **Data Loading & Cleaning** ‚Äì Combined multiple CSV sources and handled missing values.

3. **Hypothesis Testing** ‚Äì Validated assumptions using multiple statistical tests (ANOVA, T-Test, Chi-Square, Mann-Whitney U, Correlation).  
4. **Visualization & Reporting** ‚Äì Saved professional plots and generated summary tables for insights.
